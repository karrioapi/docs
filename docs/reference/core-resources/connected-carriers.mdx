The `Carrier Settings` object allow you to perform get live rates, generate labels,
track shipments and any other carrier supported operations. The API allow you to
retrieve the list of carriers configured and accessible from your account.

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```javascript
const carriers = await purplship.carriers.list();
```

</TabItem>
<TabItem value="python" label="Python">

```python
carriers = purplship.carriers.list()
```

</TabItem>
<TabItem value="php" label="PHP">

```php
carriers = purplship->carriers->list();
```

</TabItem>
<TabItem value="js" label="Javascript">

```javascript

  const carriers = await purplship.carriers.list();
```
