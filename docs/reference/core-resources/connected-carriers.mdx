The `Carrier` object allow you to perform get live rates, generate labels,
track shipments and any other carrier supported operations. The API allow you to
retrieve the list of carriers configured and accessible from your account.

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">


</TabItem>

```python
carriers = karrio.carriers.list()
```

</TabItem>
<TabItem value="php" label="PHP">

```php

```

</TabItem>
<TabItem value="js" label="Javascript">

```javascript

  const carriers = await karrio.carriers.list();
```

## Get carrier services

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```javascript
const carriers = await karrio.carriers.get_services("dhl_poland");
```

</TabItem>
<TabItem value="python" label="Python">

```python
carriers = karrio.carriers.get_services("dhl_poland")
```

</TabItem>
<TabItem value="php" label="PHP">

```php
carriers = karrio->carriers->get_services("dhl_poland");
```

</TabItem>
<TabItem value="js" label="Javascript">

```javascript
(async () => {
  const carriers = await karrio.carriers.get_services("dhl_poland");
})();
```

</TabItem>
</Tabs>
</div>

service_name as values.
