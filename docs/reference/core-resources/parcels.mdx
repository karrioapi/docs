parcels configured and accessible from your account.

## Create an Parcel


<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```javascript
const parcel = await karrio.parcels.create({
	"weight": 1,
	"weight_unit": "KG",
	"package_preset": "canadapost_corrugated_small_box"
});
```

</TabItem>
<TabItem value="python" label="Python">

```python
parcel = karrio.parcels.create({
	"weight": 1,
	"weight_unit": "KG",
	"package_preset": "canadapost_corrugated_small_box"
});
```

</TabItem>
<TabItem value="php" label="PHP">

```php
$parcel = $karrio->parcels->create([
	"weight" => 1,
	"weight_unit" => "KG",
	"package_preset" => "canadapost_corrugated_small_box"
]);
```

</TabItem>
<TabItem value="js" label="Javascript">

```javascript
(async () => {
  const parcel = await karrio.parcels.create({
    "weight": 1,
    "weight_unit": "KG",
    "package_preset": "canadapost_corrugated_small_box"
  });
})();
```

</TabItem>
</Tabs>
</div>

Throws an error if create parameters are invalid.

## Update an Parcel

Updates the specified parcel by setting the values of the parameters passed.
Any parameters not provided will be left unchanged.

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```javascript
const parcel = await karrio.parcels.update({
	"weight": 1.2,
});
```

</TabItem>
<TabItem value="python" label="Python">

```python
parcel = karrio.parcels.update({
	"weight": 1.2,
});
```

</TabItem>
<TabItem value="php" label="PHP">

```php
$parcel = $karrio->parcels->update([
  "weight" => 1.2,
]);
```

</TabItem>
<TabItem value="js" label="Javascript">

```javascript
(async () => {
  const parcel = await karrio.parcels.update({
    "weight": 1.2,
  });
})();
```

</TabItem>
</Tabs>
</div>

Returns the parcel object if the operation succeeded.
Throws an error if create parameters are invalid.

<div id="tabs">

```json
{
  "id": "pcl_99bc776deadf431a964a67c0ed8259e4",
  "weight": 1.2,

  "length": 32.0,
  "packaging_type": null,
  "package_preset": "canadapost_corrugated_small_box",
  "description": null,

}
```

</div>


## Delete an Parcel

Permanently deletes an parcel. It cannot be undone.

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```javascript
const parcel = await karrio.parcels.discard("pcl_99bc776deadf431a964a67c0ed8259e4");
```

</TabItem>
<TabItem value="python" label="Python">

```python
parcel = karrio.parcels.discard("pcl_99bc776deadf431a964a67c0ed8259e4");
```

</TabItem>
<TabItem value="php" label="PHP">

```php
$parcel = $karrio->parcels->discard("pcl_99bc776deadf431a964a67c0ed8259e4");
```

</TabItem>
<TabItem value="js" label="Javascript">

```javascript
(async () => {
  const parcel = await karrio.parcels.discard("pcl_99bc776deadf431a964a67c0ed8259e4");
})();
```

</TabItem>
</Tabs>
</div>

Returns an object confirming the operation success if the id a valid identifier was provided.

<div id="tabs">

```json
{
  "operation": "Discard parcel",
  "success": true
}
```

</div>
