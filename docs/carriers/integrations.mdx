# Karrio Carriers

Karrio allow you to access a network of shipping carriers and integrate with any
additional logistics provider.


## Carriers Connections

You can connect and use services from several carriers.

<div id="tailwind">
  <div className="grid grid-cols-1 lg:grid-cols-2 gap-6 lg:mr-8">
    {integrations.map((integration, key) => (
      <a key={key} className="group rounded-lg border-2 p-3 hover:shadow no-underline"
        href={integration.href} style={{ color: "var(--ifm-font-color-base)" }}>
        <div className="flex justify-between">
          <div className="text-base font-semibold ">{integration.name}</div>
          <code className="text-sm font-light px-2">{integration.status}</code>
        </div>

            <div>{feature}</div>

          </div>
        ))}
      </a>
    ))}
  </div>
</div>

using the Karrio API they are already familiar with.
