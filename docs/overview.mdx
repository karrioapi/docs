# Introduction

Karrio is a headless API-first shipping platform.

Karrio makes shipping services simple and accessible by offering a set of products that makes it easier
for you to add shipping functionalities to your apps.

## Getting Started

The easiest way to give karrio a whirl, apply to join our beta at [karrio.io](https://karrio.io).
Otherwise, you can follow [installation guide](/installation) to host karrio where you want.

The karrio server is in Python, but you can use any programming language to send API requests
to all supported shipping carriers through karrio.
