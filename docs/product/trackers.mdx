The karrio server exposes a shipment' tracker API that can watch
and store the latest known status of a package during the delivery
journey.

:::info

Note that a tracker is automatically created for all shipments created
using the karrio API.

:::

## Track a shipment

You can add a tracker to any shipment from the registered carrier accounts
whether they were created using karrio or not.

:::note To track a shipment:

1. Go to the `Trackers` section.
2. Click `Track a Shipment`.
3. Enter your tracking number.
4. Select the related carrier account.
5. Click `Submit`.

:::

## Preview Tracking Details

The karrio server tracking API give you access to the full shipment
tracking status.

:::note To track a shipment:

1. Go to the `Trackers` section.
2. Click `Track a Shipment`.
3. Enter your tracking number.
4. Select the related carrier account.
5. Click `Submit`.

:::

:::info

The karrio dashboard comes with a basic branded tracking page.
You can always customize it to fit your needs.

:::

:::caution

We currently do not support a tracking page editor for in-app
branding customization.

:::
