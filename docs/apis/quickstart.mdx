# Development quickstart

Get up and running with our client libraries and start developing your Karrio
integration.

Integrating Karrio into your app or website can begin as soon as you create
a Karrio account, and requires three steps:

`1` [Install a client library](#1-install-a-client-library) so your integration
can interact with the Karrio API

`2` [Obtain your API keys](#2-obtain-your-api-keys) so Karrio can authenticate
your integration’s API requests

`3` [Make a test API request](#3-make-a-test-api-request) to confirm everything
is up and running


---


## `1` Install a client library

We provide official libraries for different programming languages.

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```shell
# Install via npm
npm install --save karrio
```

</TabItem>
<TabItem value="python" label="Python">

```shell
# Install through pip
pip install --upgrade karrio-python
```

</TabItem>
<TabItem value="php" label="PHP">

```shell
# Install the PHP library via Composer
composer require karrioapi/karrio-php
```

</TabItem>
<TabItem value="js" label="Javascript">

```shell
# Add script to an HTML page
<script src="https://api.karrio.io/static/js/karrio.js"></script>
```

</TabItem>
</Tabs>
</div>

## `2` Obtain your API keys

Karrio authenticates your API requests using your account’s API keys.
If you don’t include your key when making an API request, or use one
that is incorrect or outdated, Karrio returns an error.

Karrio accounts support two type of keys: an `API key` ideal for server
applications and a `JSON Web Token` ideal for client applications.

You can generate new `JSON Web Token` by API requests providing an account
credentials (email, password). For convenience, your `API key` is
always available in the Dashboard

## `3` Make a test API request

To check that your integration is working correctly, make a test API request
using your `API key` to create an Address.

<div id="tabs">
<Tabs>
<TabItem value="node" label="Node">

```javascript
// Set your API key.
// See your keys here: https://app.karrio.io/developers/api
const karrio = require('karrio')('key_...');

const adddress = await karrio.addresses.create({
	"address_line1": "125 Church St",
	"person_name": "John Doe",
	"city": "Moncton",
	"country_code": "CA",
	"postal_code": "E1C4Z8",
	"state_code": "NB",
});
```

</TabItem>
<TabItem value="python" label="Python">

```python

# See your keys here: https://app.karrio.io/developers/api
import karrio
karrio.api_key = 'key_...'

adddress = karrio.addresses.create({
	"address_line1": "125 Church St",
	"person_name": "John Doe",
	"city": "Moncton",
	"country_code": "CA",
	"postal_code": "E1C4Z8",
	"state_code": "NB",
});
```

</TabItem>
<TabItem value="php" label="PHP">
