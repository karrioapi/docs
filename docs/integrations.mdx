# Karrio Carriers

Karrio allow you to access a network of shipping carriers and integrate with any
additional logistics provider.

## Carriers Connections

You can connect and use services from several carriers.

## Generic Carrier

Karrio allow you to define your own carrier. This is often use to manage carriers
without APIs under a single platform. This also enables small carriers without API
to use Karrio as their API.
